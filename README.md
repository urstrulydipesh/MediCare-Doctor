# 🏥 MediCare - Doctor Appointment Booking System

A full-stack MERN web application that allows patients to browse and book appointments with doctors, make secure payments, and leave reviews. Doctors can manage their profiles and view scheduled appointments.

## ✨ Key Features

- **Browse & Search Doctors** - Find doctors by specialty and ratings
- **Book Appointments** - Easy appointment scheduling system
- **Secure Payments** - Integrated payment gateway for bookings
- **User Dashboards** - Separate dashboards for patients and doctors
- **Leave Reviews** - Rate and review doctors
- **Secure Authentication** - JWT-based login and registration
- **Responsive Design** - Works on all devices

## 🛠️ Tech Stack

**Frontend:** React, Vite, Tailwind CSS, React Router  
**Backend:** Node.js, Express, MongoDB, JWT  
**Deployment:** Vercel (Frontend), Render (Backend)

## 🚀 Quick Start

### Clone Repository
```bash
git clone https://github.com/yourusername/Doctor-Appointment-Booking-MERN-Stack.git
cd Doctor-Appointment-Booking-MERN-Stack
```

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
